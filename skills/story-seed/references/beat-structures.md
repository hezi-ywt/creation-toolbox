# 节拍结构参考

> 给 AI 用的。决定要不要给创作者建议某种节拍结构、以及怎么用。每种结构看完应该 30 秒内能判断"用 / 不用 / 怎么改"。

---

## 1. Save the Cat 15 节拍

Blake Snyder 写的电影长片节拍表，按 110 分钟剧本来分配。下面百分比按总片长算。例子用《肖申克的救赎》贯穿。

**1. 开场镜头（0-1%）**
给主角"之前"的世界定一个画面，跟结尾镜头形成对照。一个静止的、有信息量的画面，告诉观众主角现在过得是什么样的日子。
例：法庭上，安迪被宣判。

**2. 主题陈述（5%）**
配角随口说一句话，把全片要回答的命题埋下去。主角当时听不进去，到结尾才会懂。
例：瑞德说"希望是危险的东西"。

**3. 设定（1-10%）**
主角的日常 + 他缺什么 + 三个"需要修复"的地方（性格弱点、关系裂痕、外部困境各埋一个）。
例：安迪是银行家、被冤枉、对妻子冷淡——三块都埋。

**4. 催化剂（10-12%）**
一个事件把主角踢出舒适区。不可逆——回不去了。
例：安迪入狱。

**5. 争论（12-25%）**
主角犹豫："我到底要不要接这个挑战？"这段是给观众建立 stakes 的，不是浪费。
例：安迪适应监狱、被狱警和"姐妹花"欺凌、想过自杀。

**6. 第二幕开始（25%）**
主角**主动**选择进入新世界。注意：必须是主动选择，被推进去不算。
例：安迪决定"忙着活，或者忙着死"。

**7. B 故事（25-30%）**
一条副线开始，通常是关系线，专门用来承载主题。主角最后是靠 B 故事的领悟才能解决 A 故事。
例：安迪和瑞德的友谊。

**8. 欢乐时光（30-55%）**
"答应观众的爽点"——预告片素材都在这里。前提（"如果一个银行家进了监狱会怎样"）的回报。
例：用税务知识帮狱警省钱换啤酒、建图书馆、广播放歌剧。

**9. 中点（50%）**
假胜利或假失败。无论哪种，赌注都被升级——后果变得更严重。
例：典狱长开始利用安迪给自己洗钱（看似胜利，实际是更深的笼子）。

**10. 坏家伙逼近（55-75%）**
反派内外夹击。外部反派加压，主角内心也开始崩，盟友之间出现裂痕。
例：汤米被杀以堵住安迪翻案的可能，安迪被关禁闭两个月。

**11. 失去一切（75%）**
最低点。一定要有"死亡气息"——有人真的死、梦想死、旧身份死，三选一。
例：汤米的死、翻案希望彻底破灭。

**12. 灵魂黑夜（75-85%）**
主角独自面对自己，把主题想透。这段是内心戏，外部动作几乎停滞。
例：安迪和瑞德谈"希望"和"忙着活"。

**13. 第三幕开始（85%）**
B 故事的领悟让主角找到了答案。新计划开始。
例：安迪宣布"如果出去了，要去墨西哥"——他其实已经决定执行越狱。

**14. 终场（85-99%）**
主角用学到的东西解决问题。反派倒下。这里通常是 5 个节拍组成的迷你结构（集合团队 → 行动 → 高塔意外 → 挖深井 → 终局）。
例：越狱 + 用账本扳倒典狱长。

**15. 结尾镜头（99-100%）**
跟开场镜头对照，证明世界变了 / 主角变了。
例：安迪和瑞德在墨西哥海边重聚。

**何时用 / 何时别用**
用：电影长片（90-130分钟）、商业片、类型片、第一次写长片的人。它最大的好处是给"中点"和"失去一切"两个钉子，防止第二幕散掉。
别用：作者电影、慢节奏文艺片、群像戏、3 集以内的短内容——硬套会很匠气。

## 2. 三幕剧

最古老最通用的结构。三千年没人推翻。

| 幕 | 占比 | 任务 | 关键节点 |
|---|---|---|---|
| 第一幕 | 0-25% | 建立：主角是谁、世界是什么、想要什么 | 开场（钩子） / 第一转折点（25%，把主角推进主线） |
| 第二幕 | 25-75% | 对抗：主角不断尝试、不断受挫、被迫成长 | 中点（50%，赌注升级 or 真相揭示） / 第二转折点（75%，最低谷或重大转变） |
| 第三幕 | 75-100% | 解决：决战 + 答案揭晓 + 新平衡 | 高潮（85-95%） / 结尾（100%） |

第二幕是最容易写垮的——它占一半篇幅，但只靠"尝试-失败"循环撑着。Save the Cat 之所以流行，就是因为它把第二幕的中点钉死了。

**何时用 / 何时别用**
用：任何长度都能套（90 分钟电影、12 集剧、3 万字小说），尤其当你只想要个最粗的骨架不想被细节限制。
别用：如果创作者已经在用 Save the Cat 或英雄之旅，别叠加——三幕剧是更高一层的抽象，叠了就重复。

## 3. Pixar Pitch 6 句模板

皮克斯故事板 Emma Coats 总结的"一句话故事公式"。中文版：

> 从前，有一个 ____。每天，____。直到有一天，____。因为这样，____。因为这样，____。最后，____。

6 句话里前 2 句是日常 + 缺失，第 3 句是催化剂，4-5 句是连锁后果（注意是"因为这样"不是"然后"——强调因果），最后是结局。

**例 1（老题材：《海底总动员》）**
- 从前，有一个过度保护的小丑鱼爸爸马林。
- 每天，他把儿子尼莫看得死死的，怕外面的危险。
- 直到有一天，尼莫被潜水员抓走了。
- 因为这样，马林被迫独自横渡海洋去救他。
- 因为这样，他在路上学会了信任陌生人、放手让儿子长大。
- 最后，他找到了尼莫，也学会了不再把他锁在身边。

**例 2（新题材：AI 时代下岗的配音演员）**
- 从前，有一个配音了 20 年广告的中年男人老陈。
- 每天，他对着话筒念稿，养活老婆和上大学的女儿。
- 直到有一天，公司换了 AI 配音，他被裁了。
- 因为这样，他偷偷接私活，给一个独立游戏配反派。
- 因为这样，他第一次演了不是为了钱的角色，发现自己其实想当演员。
- 最后，他没回到广告行业，去了一个小剧场——挣得少，但终于是他自己的声音。

**何时用 / 何时别用**
用：种子阶段，故事还很模糊；创作者卡壳，需要快速看到"完整一条线长什么样"；用来测试一个 idea 有没有戏（如果 6 句填不出来，说明因果链断了）。
别用：故事已经成型，再用这个会被"6 句"框住，丢掉细节。

## 4. 英雄之旅（简化版）

Joseph Campbell 12 阶段太学院派，Christopher Vogler 给好莱坞用的版本是 12 阶段，下面再压到 8 个最有用的：

1. **普通世界**——主角是谁，缺什么。和 Save the Cat 的"设定"等价。
2. **冒险召唤**——一个机会/危险出现，要求主角行动。
3. **拒绝召唤**——主角说"不、我不行、我没空"。
4. **遇到导师**——一个比他懂的人给他工具、信息或者只是一脚。
5. **穿越门槛**——离开熟悉世界，进入新规则的领域。从这里开始不能回头。
6. **试炼 / 盟友 / 敌人**——一连串小考验，结识同伴和对手，看清新世界的规则。
7. **最深洞穴 / 苦难**——直面最大的恐惧或失去，"象征性死亡"。
8. **回归**——带着新东西回到旧世界，旧世界被他改变 or 他改变了旧世界。

**何时用 / 何时别用**
用：成长故事、奇幻 / 冒险 / 武侠 / 修仙、游戏剧情、有明确"出门—回家"轨迹的故事。
别用：现代故事很少**完全**套用，因为它的形而上学色彩太重（"召唤""门槛""导师"很容易写得假）。但**部分挪用**很常见——比如只借"拒绝召唤"来解释主角为什么一开始犹豫，或者只借"最深洞穴"来设计低谷。AI 给建议时优先推荐挪用 1-2 个阶段，不要整套塞。

## 5. 短剧节奏（中国市场 2026 当红）

竖屏短剧的节拍跟传统影视完全不一样。下面是 2026 年仍然管用的实操规则。

**总体框架**
- 单集 1-3 分钟，全剧 80-120 集（行业平均 100 集左右）
- 单集播放完播率是命脉，所以**开场 5 秒内必须有钩子**（钩子 = 冲突、悬念、爽点、反差、未解之谜）
- **平均 3 集一反转**——节奏远高于传统剧。一旦超过 4 集没反转，用户就划走了
- 爽点 / 虐点 / 误会点 三种情绪轮转，不能连续两集同一种

**单集结构（90 秒版）**
- **0-10 秒：开钩**——把上集结尾接住，或者扔一个新冲突直接进场
- **10-50 秒：加码**——情绪堆叠，主角被进一步逼到墙角 or 进一步爽
- **50-75 秒：反转**——一个揭穿、一个打脸、一个身份反转、一个误会反转
- **75-90 秒：留扣**——一个新冲突 / 新人物 / 新信息抛出来，强制看下一集

**有效的钩子类型**（用在开场 5 秒和单集留扣）
- 身份反转：清洁工其实是首富 / 弱鸡儿媳其实是隐世大佬的孙女
- 当场打脸：被嘲讽的下一秒揭穿身份
- 极限误会：男主撞见女主和别的男人，转身就走（误会要在 1-3 集内解开）
- 灾难倒计时：还有多少集要离婚 / 还有多少天要死
- 致命悬念：尸体出现但凶手身份留到下一集

**男频 vs 女频**
- **男频**：核心是**逆袭爽感**。基本节奏 = 主角受辱 → 打脸 → 升级 → 更大的辱 → 更大的打脸，螺旋上升。题材常见：战神/赘婿/重生/权谋/兵王/医仙/神医。爽点密度比女频更高，虐点占比更低。典型套路：上门赘婿被全家瞧不起 → 神秘电话来了 → 岳父跪下叫"集团总裁"。
- **女频**：核心是**情感拉扯**。基本节奏 = 误会 → 虐 → 解开 → 甜 → 新的误会。题材常见：豪门/契约婚/失忆/重生/虐恋/双向暗恋/替身。虐点和误会点占比高于男频，爽点常以"男主反向追妻 / 反派打脸"的形式出现。典型套路：女主被陷害离婚 → 男主才发现真相 → 跨越千山万水追妻。

**何时用 / 何时别用**
用：创作者明确说要做短剧、竖屏、几分钟一集、要投流量到抖音/快手/YouTube Shorts。短剧不是"短的剧"，是另一个物种，节拍逻辑跟电影完全不同。
别用：长片、网络剧、剧场作品。如果创作者想要"细腻的情感"或"留白"，明确告诉他短剧不允许这种东西——你想要那个，那就不是短剧。

## 决策表

| 创作者说 | 推荐 |
|---|---|
| "我要写电影长片" | 三幕剧打骨架 + Save the Cat 钉中点 |
| "想做短剧 / 竖屏 / 抖音剧" | 短剧节奏（别推 Save the Cat） |
| "故事走向我没想清楚，先快速搭个骨架" | Pixar Pitch（6 句填空） |
| "我有一个 idea，不知道能不能成立" | Pixar Pitch（填不出来就说明因果链断了） |
| "想写成长故事 / 奇幻 / 武侠" | 英雄之旅（整套或挪用 1-2 阶段） |
| "我的故事第二幕老是散" | Save the Cat 的"中点"和"失去一切" |
| "网剧 / 长剧集（12-24 集）" | 三幕剧 + 每集小三幕 |
| "短篇小说 / 网文单本" | 三幕剧 + Pixar Pitch 当大纲 |
| "我就想要一句话讲完这个故事" | Pixar Pitch 的第 3 句（催化剂那句） |
| "作者电影 / 文艺片 / 慢节奏" | 别推 Save the Cat，三幕剧最多，鼓励他自己定节奏 |
| "群像戏 / 多主角" | 别推上面任何一个套整套，建议每个主角各自走自己的小弧线 |
