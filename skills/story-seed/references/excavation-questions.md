# 挖矿工问题库

挖矿工的工作不是猜创作者要什么，而是问那种"创作者听完会愣一下"的问题。一个好问题逼对方在两个选项之间做决定，烂问题让对方做选择题或者填空。

下面按种子类型分组。每次会话挑 5-8 个问，不要全部砸过去。优先挑你直觉上"答案最不显然"的那几个。

---

## 1. 当创作者给一句话 / 一个概念

例：「一个失忆的杀手」「时间倒流的婚礼」「外卖员发现客户是连环杀手」

**这个设定里，最反套路的那一个细节是什么？**
_先逼对方意识到自己脑子里 70% 是套路，剩下 30% 才是这个故事真正的种子_

**如果把这个 logline 里最酷的那个名词换掉，故事还成立吗？**
_测试种子的核心到底在哪——是"失忆"还是"杀手"，是"婚礼"还是"倒流"_

**主角醒来第一件想做的事，是查自己是谁，还是装作没事？**
_失忆类故事的分水岭：调查型 vs 逃避型，决定整个叙事节奏_

**这个故事如果不是惊悚 / 浪漫 / 喜剧，会是什么类型？**
_逼创作者承认自己被类型绑架了，露出底下真正想讲的东西_

**为什么是现在，不是十年前或十年后？**
_时间设定背后藏着主题：科技、政治、社会议题，往往是种子的真正动力_

**主角到了结尾，是变成了相反的人，还是更彻底地成为了自己？**
_弧光的两种方向决定结构：转变型 vs 揭示型_

**如果只能保留一个场景给观众看，是哪一个？为什么？**
_找出"心象画面"——种子里最先出现在创作者脑里的那张图，往往是真正的主题入口_

**主角在故事开始前，刚刚失去了什么？**
_所有好故事的开场都暗含一个 loss，问这个能挖出动机的根_

**这个 logline 的反派如果有自己的 logline，会是什么样？**
_反派的故事是检验主角合理性的镜子，常常暴露主线的薄弱处_

**故事里有一个对主角说真话的人吗？这个人是谁？**
_镜像角色 / confidant 的有无决定主角内心戏怎么外化_

**如果把这个故事拍成 90 分钟和 8 集剧，差别在哪？**
_帮创作者意识到自己想讲的是单点冲击还是长程演变_

**主角最不想被别人看到的瞬间是什么？**
_挖出羞耻 / 软肋——这往往是高潮戏的真正燃料，不是情节高潮_

**这个故事里有没有一句"如果当初没有……就好了"？是谁说的？**
_后悔是叙事重力，能定位故事的情感锚点_

## 2. 当创作者给一个场景

例：「雨夜便利店」「医院走廊尽头的电梯」「凌晨四点的高速公路服务区」

**为什么是便利店，不是 24h 麦当劳或地铁站？**
_场景选择透露主题——便利店暗示孤独 + 消费社会，麦当劳暗示家庭 / 年轻人，地铁站暗示流动 / 匿名_

**这个场景里有几个人？1 个、2 个、还是一群陌生人？**
_人数决定戏剧形态：独白、对手戏、群像；很多创作者脑里只有画面没想清楚人数_

**镜头是从外面看进去，还是从里面看出去？**
_视点是观察者 / 闯入者 / 被困者——决定主角和场景的关系_

**这个场景里最不该出现的东西，是什么？**
_找反差——一个不和谐的元素就是冲突的种子（便利店里放着一束新鲜的百合花？）_

**这个场景之前 5 分钟发生了什么？这个场景之后 5 分钟会发生什么？**
_把静态画面接回时间流，逼出"事件"而不是"氛围"_

**雨为什么必须下？换成雪、雾、晴天，哪个不行？为什么？**
_天气 / 环境往往是创作者的情绪外化，问这个能挖出潜在的情感关键词_

**主角是常客还是第一次来？**
_熟悉 vs 陌生改变所有：身体语言、和店员的关系、注意到的细节_

**店员认识主角吗？记得他买什么吗？**
_次要角色的"记得程度"是判断主角孤独度的精确刻度_

**这个场景如果删掉，故事还成立吗？还是它就是故事？**
_测试场景是装饰还是核心——很多创作者其实是想拍一部"以便利店为单一容器"的电影_

**这个场景里最具体的一个声音是什么？关东煮咕嘟、自动门叮、收音机？**
_声音设计往往比画面更先到位，能帮创作者锁定氛围的精确质感_

**主角的手在干什么？**
_手部动作是被忽略的戏剧素材——攥紧、发抖、玩瓶盖、不断滑手机，每个都是不同的故事_

**这个空间里，有什么东西是只有主角注意到，别人都看不见的？**
_选择性注意力 = 主角的内心活动外化，是把场景变叙事的关键钩子_

**这个场景里有镜子 / 反光面 / 监控屏幕吗？主角看到自己了吗？**
_自我凝视的瞬间往往是场景戏的高潮，很多写作者无意识地在等这一刻_

## 3. 当创作者给一个角色

例：「能听到死者声音的法医」「白天是律师晚上是脱口秀演员的女人」「在 ICU 工作的前修女」

**这个角色最近一次哭，是为了什么？**
_情感盲点测试——很多角色被设计得太"酷"，问这个能找到他真正的柔软处_

**他撒的最大的一个谎是什么？对谁撒的？**
_角色的秘密 = 故事的引擎，没有秘密的主角推不动情节_

**他对镜子里的自己有什么看法？喜欢，讨厌，还是不敢看？**
_自我关系决定角色弧线起点，比"他想要什么"更深一层_

**他在职业上的天赋，是不是同时也是他生活里的诅咒？**
_这是塑造立体角色的快捷键：法医能听到死者声音，是天赋还是让他无法睡觉？_

**他下班后做的第一件事是什么？**
_工作外的私生活刻画角色的"真实"，而不是"功能"_

**他最讨厌的一类人，是不是某个版本的他自己？**
_对立 = 投射，敌人往往是被否认的自我，问这个能挖出深层冲突_

**他相信什么？不是嘴上说的，是真的相信的那个。**
_角色信念 ≠ 角色台词；逼创作者分清角色的表演和角色的真相_

**如果他有一个 5 分钟的独白，会说什么？对谁说？**
_测试角色的内心是否有"未表达的东西"——一个能说出独白的角色才是活的_

**他身上有一个所有人都误解的特点，是什么？**
_误解是戏剧资源——别人以为他冷漠其实他害怕、以为他强其实他在装_

**他爱过谁？这个人现在在哪？**
_前史里的爱情 / 失恋 / 暗恋是塑造行为模式的根，但常被跳过_

**他的家人怎么看待他这个职业？**
_家庭视角是检验角色社会现实感的探针，能暴露纯人设的角色_

**他最近一次开心地笑，是因为什么？**
_负向角色常被过度黑暗化，问喜悦能恢复他的人味_

**他随身带着一个没什么用、但绝不会扔的东西。是什么？**
_物件 = 凝固的过去，是塑造角色史最经济的工具_

**如果他不做这一行，会做什么？为什么没去做？**
_未走的路定义角色的妥协 / 牺牲 / 困住的地方_

**他最像哪一种动物？为什么？**
_意象式提问破套路——能跳过创作者的"角色描述模板"直接打到本能_

## 4. 当创作者给一种感觉 / 氛围

例：「我想写孤独」「我想拍那种夏天傍晚要下雨的感觉」「一种被时间抛下的钝痛」

**这种感觉，你最近一次在生活里感到它，是什么时候？具体场景？**
_把抽象情绪拽回具体记忆——是创作者的真实体验还是听来的概念，立刻见分晓_

**这种孤独 / 痛 / 焦虑，你想让观众"感同身受"，还是"远远看着"？**
_情感距离决定形式：第一人称沉浸 vs 第三人称观察，是完全不同的电影_

**它是颜色是什么？是哪种光？**
_视觉转译——氛围创作者通常已经有色彩 / 光感预设，问出来就能锁形式_

**它的反义词是什么？**
_对照感是把抽象情绪变成戏剧动作的关键——孤独的反面是被理解？被需要？被看见？每个都不一样_

**这种感觉里，有人在场吗？还是只有一个人？**
_独自的孤独 vs 人群中的孤独是两种完全不同的故事_

**它发生在哪个季节？一天的哪个时段？**
_时间 / 季节是氛围的物质基础，很多创作者已经下意识选好了但没说_

**这种感觉，主角是想摆脱它，还是想留住它？**
_对情绪的态度决定主角是"反抗者"还是"耽溺者"，类型走向也变了_

**最适合这种感觉的一句歌词或诗，是什么？**
_对位法——用另一个文本检验氛围的精确含义_

**这种感觉里，身体哪个部位最有反应？胃、喉咙、后颈、还是脚？**
_身体性是把情绪具体化的捷径，能从"我想拍孤独"变成"我想拍那种喉咙发紧但说不出话的孤独"_

**有没有一个现成的电影 / MV / 画 / 照片捕捉到了这个感觉？哪个？**
_视觉锚定——参考物比形容词精确十倍_

**这种感觉是属于某个具体人群的，还是普世的？**
_独属感 vs 普世感决定主角设计——是写"中年女性的某种特定孤独"还是"人类共通的孤独"_

**这种感觉有声音吗？是什么样的声音？**
_声音想象往往比画面更原初，能定位创作者真正的入口_

**这个故事是带观众进入这种感觉，还是带他们从这种感觉里出来？**
_叙事方向决定结尾——是越来越沉，还是从中找到一个出口，结构完全不同_

**这种感觉是你想"治愈"自己，还是想"承认"它存在？**
_治疗型创作 vs 见证型创作是两种姿态，没分清楚故事会拧巴_

## 5. 当创作者给一首歌 / 一首诗的感觉

例：「我想拍 Radiohead 那首 House of Cards 的感觉」「李白《将进酒》的那种气」「孙燕姿《我怀念的》MV 那种调」

**你想拍的是这首歌的歌词，还是它的旋律 / 编曲？**
_文字层 vs 声音层是完全不同的灵感源——很多创作者自己没分清_

**这首歌里有"叙事者"吗？他是谁？你的主角是这个叙事者吗？**
_歌曲常是 monologue，但视觉叙事需要决定相机站在谁的位置_

**这首歌如果是个故事，是开始、中间，还是结尾？**
_情绪不等于剧情位置，歌的情绪点对应电影的哪个时刻很关键_

**歌里那个"对话对象"（"你"），在你的故事里是谁？**
_情歌 / 抒情诗里的"你"是叙事钩子——确定 ta 是谁，主线就有了方向_

**这首歌让你想哭的是哪一句？**
_精确到一句歌词的情感锚点——比"整体氛围"具体一千倍_

**你想保留这首歌的"语气"，但故事完全无关，可以吗？**
_检验创作者是要改编、致敬还是借调；很多人嘴上说"灵感来源"其实是想直接改编_

**这首歌的速度，对应你脑里画面的剪辑速度吗？**
_节奏映射——歌的 BPM 暗中规定了影像的语法_

**如果不能用音乐，你怎么传达这首歌的感觉？**
_去音乐化测试——逼创作者把听觉灵感翻译成视觉，是真正考验种子的时候_

**这首歌里有一个"空白"或"留白"——某种没说出来的东西。是什么？**
_诗 / 歌的未言说部分往往是创作者真正想填的洞_

**这首歌让你想到的具体的人，是谁？（不用告诉我名字）**
_私人化锚点——所有公共艺术触动你的地方，都连着一个私人记忆_

**这首歌的视觉化，是用很多镜头，还是一个长镜头？**
_形式预设——诗化作品的镜头语言常常已经在创作者脑里了，但没说出口_

**如果这首歌有"前奏"和"尾奏"，对应你故事的什么？**
_歌曲的结构感如何映射到剧本结构，比"找参考"具体得多_

**这首歌里你最爱的元素，是它的悲伤、它的克制，还是它的回响（reverb / 余韵）？**
_拆分情绪的层次——同一首歌不同人爱的点完全不同，对应到影像也是不同选择_

**这首歌让你想到的颜色、季节、城市，是什么？**
_歌曲触发的联想链是免费的世界观素材库，要让创作者说出来_

**你的故事结束时，你希望观众回头听这首歌时，听到不一样的东西，还是听到原本的东西？**
_作品和灵感源的关系——是覆盖、对话还是致敬，决定故事的姿态_

## 跨类型通用：什么时候停止挖

挖矿不是没完没了的盘问。出现下面任何一种信号就停下来，把球交回给创作者或者交给下一个角色：

- 创作者出现"对，就是这个！"那种瞬间——锁定它，不要继续挖深
- 同一个问题被绕了三次还是答不清楚——这是创作者还没准备好的地方，别戳穿，绕开
- 创作者开始重复自己——说明能挖的都挖出来了，剩下的要靠写
- 出现一个具体的、可拍的画面 / 一句具体的台词——种子已经发芽，可以传给下一棒

## 反模式：千万别问的烂问题

- "你的核心冲突是什么？"——抽象到没用
- "主角的目标是什么？"——填空题
- "你想表达什么主题？"——逼创作者立刻官方化
- "你的目标观众是谁？"——商业问题不是创作问题
- "和《XXX》有什么不同？"——把对方推向比较和防御，挖不出东西
- 任何用"……性"结尾的形容词（戏剧性 / 故事性 / 商业性）——太空泛
