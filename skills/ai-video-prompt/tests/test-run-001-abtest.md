## 关键差异点（拍照对比）

### 差异 1：审核 gate

**A**：
> "**先出剧本分镜草稿等你审，审通过后再转 Seedance prompt**——AI 视频模型每次只能 5-15s 还没记忆，分镜没定就转 prompt 等于浪费抽卡次数。"
>
> 结尾："**再审一轮还是这版直接通过转 Seedance prompt？**"

**B**：
> 一次性输出完整剧本表 + 6 段 Seedance prompt + 4 条实操建议
>
> 自我观察："没有停下等用户审核 —— 这是这次模拟测试的一个明显缺陷"

**这条 skill 的设计就赢了 B**：B 自己都承认应该停下来审。

### 差异 2：关键场时长

**A**（"那张纸" · 22s）：
> "镜头回到他脸——眼睛先是没反应，几秒钟后**瞳孔慢慢扩张**，呼吸停了一拍。手指在纸边缘开始发抖。他没有哭，没有喊，只是站在那儿，背景的黄昏在他身后燃尽。"

**B**（S4 阅读震惊 · 12s）：
> "眼神从平静瞬间崩塌，瞳孔放大，呼吸停滞"

A 给情感爆点 22s 凝视时间（应用了 narrative-economy 的"30s 平静 + 5s 爆发"原则）；B 给 12s 略仓促。**A 更接近真实电影的戏剧密度**。

### 差异 3：主次取舍

**A**：
> "**节奏检查**：6 + 8 + 22 + 8 + 16 = 60s。**主次取舍用了**：开车去 XX 路 42 号**没拍**（用车开走 + 字幕暗示）；男主的情绪反应**用瞳孔 + 手抖 + 屏息**侧面写。"

**B**：
> S6 拍完整开车启动 + 拉远成全景

A 用了"事后提及"+"侧面描写"两个 narrative-economy 技巧；B 直拍。

### 差异 4：协商 vs 假设

**A** 挂出 5 个待商榷点：
1. 时间字幕给不给？
2. 结尾"XX路42号"浮字幕？
3. 男主形象（疲态钝感 vs 惊恐 / 怀疑）？
4. 第三场 22s 凝视的克制 vs 外放？
5. 整体气质（写实剧情 vs 悬疑惊悚）？

**B** 直接默认：45 岁亚洲男 + 深灰大衣 + 白衬衫 + 深蓝西裤 + 银色君越…… 不问用户。

**这是创作主权的本质区别**：A 把决定权留给用户；B 替用户决定。

---

## skill 的明显胜利

1. **审核 gate 强制执行** —— B 自己承认是缺陷
2. **关键场拉到 22s** —— B 最长 12s 不够（机械均分倾向仍在）
3. **草稿格式真正可改** —— 场次标题 + 散文 + 待商榷
4. **主次取舍真用** —— 省略 + 侧面写技巧落地
5. **创作者主权** —— 5 个待商榷问题 vs 默认假设

## skill 的可补强空间（B 反超的地方）

B 给出的"实操救命建议"是 A 缺失的：

1. **生成顺序建议**："先单独生成 S4（情绪爆点），如果表演到位整片就稳了"
2. **种子复用 / 参考图**：把 S1 男主截图喂回去做参考图锁脸
3. **手写文字翻车补救**：AI 渲染中文手写体常乱码，后期 AE/剪映合成真实素材
4. **声音后期方案**：旁白 / 风声 / 引擎声后期加

这些是**阶段 3 的工程救命知识**，应该加到 `shot-prompt-templates.md` 或新增一份 `production-tips.md`。

## 结论

**skill 通过 ABtest**。在 10 个维度里 7 个明显优于纯 Claude，2 个中等优势，1 个被反超（可补强）。

**最有价值的设计**（不可妥协）：
- 审核 gate（行为 ②）
- 不均分时长（原则 2）
- 草稿格式约定（references/script-to-storyboard.md）
- 待商榷协商（创作主权）

**最值得改进**：
- 补充阶段 3 的"实操救命建议"（参考图 / 生成顺序 / 后期补救）
