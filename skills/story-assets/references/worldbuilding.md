# 世界观参考

世界观**为故事服务**，不是为了"显得宏大"。**写多少取决于故事真用到多少**——日常剧只要"现代上海"够了，玄幻必须搭整套魔法系统。

## 4 个维度（按"必要性"排，不是必填）

### 1. 规则（最核心 · 决定能发生什么）
- **物理规则**（科幻 / 玄幻必须）：魔法系统 / 修真等级 / 时空规则
- **科技水平**（科幻 / 都市必须）：手机 / 电车 / 火枪 / 蒸汽
- **超自然规则**（悬疑 / 恐怖必须）：鬼魂 / 异能的"边界"在哪
- **日常逻辑**（写实剧必须）：主角能不能随便辞职 / 去医院多久能挂上号

### 2. 社会（决定主角能做什么）
- 政治体制 / 阶层 / 权力结构
- 经济（主角怎么吃饭 / 多有钱 / 房子怎么来的）
- 文化（节庆 / 婚丧 / 禁忌 / 流行）
- 教育 / 宗教（如果涉及）

### 3. 历史（决定主角从哪来）
- 关键历史事件（影响现在的）
- 时间线（如果故事跨年代）
- 集体记忆（角色们都知道的事）

### 4. 地理（决定主角能去哪）
- 主要场所（城市 / 村 / 学校 / 大本营）
- 场所之间的距离 / 交通
- 标志性地标（情感意义的）

## 按题材的繁简取舍

| 题材 | 必须做 | 可省略 | 例 |
|---|---|---|---|
| **日常 / 都市** | 社会（阶层 + 工作）| 规则 / 历史 | 不用写"21 世纪有手机"|
| **悬疑 / 推理** | 规则（凶手能做什么）+ 地理（案发地）| 大历史 | 重点是空间 |
| **玄幻 / 仙侠** | **规则（修炼体系）+ 社会（门派阶层）+ 历史**| 不太能省 | 三者打底，缺一垮 |
| **科幻** | **规则（科技 / 物理）+ 历史（怎么变成这样）**| 日常细节 | 设定厚重 |
| **言情 / 偶像** | 社会（背景 / 阶层差）+ 关系网 | 规则 / 历史 | 重情感，背景虚化 |
| **历史 / 古装** | 历史 + 社会（朝代 / 礼制）| 自创规则 | 用真实历史省事 |
| **童话 / 奇幻** | 规则（魔法）+ 地理（奇幻地图）| 写实社会 | 想象优先 |

**核心判断**：**写出来的设定要在场景里被用到**——光设定不用的"附录"是死设定，删掉。

## 完整示例（日常剧 · 极简版）

```markdown
---
title: 时代背景
type: 世界观
status: 已定
tags: [2025, 上海, 广告业]
created: 2026-05-17
modified: 2026-05-17
---

# 时代背景

## 时空
- 2025 年的上海，主体故事在内环外、外环内的"夹缝带"
- 时间跨度：3 个月（仲秋 → 入冬）

## 社会
- 广告业正在被 AI 大幅冲击，主角的 4A 公司去年裁了 30%
- 主角属"还没被裁但人人自危"的中层
- 房价稳定但市场冷清，房贷压力依然在

## 关键细节
- 男主的车是 2018 年的老款（暗示经济没起色）
- 女儿上的是公立学校（不是国际学校 → 阶层定位）
- 整片**没有任何超自然元素**，所有"反常"都要现实解释

## 不做的事
- 不写国际局势（跟故事无关）
- 不深入广告业内幕（背景，不是主角领域）
```

## 完整示例（玄幻 · 中量版）

```markdown
---
title: 修真体系
type: 世界观
status: 已定
tags: [玄幻, 修真, 灵气, 阶层]
created: 2026-05-17
modified: 2026-05-17
---

# 修真体系

## 修炼阶段（核心规则）
1. **凝气**（1-9 层）：日常人能修，开窍即入门
2. **筑基**（前 / 中 / 后期）：人间精英，可活 200 岁
3. **金丹**：进入"修士"行列，可独立门户
4. **元婴**：宗门长老级
5. **化神**：传说级，万年一出

## 灵气分布（决定地理）
- **灵气浓郁带**：东海三岛 / 北方雪山 / 西域沙海（资源争夺战发生地）
- **灵气贫瘠带**：中原大陆（凡人 90% 在此）
- **绝灵之地**：南方瘴林（连凝气都修不了，被流放者去处）

## 门派阶层
- 五大宗（化神坐镇）→ 一线门派（元婴）→ 二三线 → 散修
- 阶层固化严重，平民想入大宗几乎不可能（除非天才）

## 禁忌 / 规则
- 杀同门 = 被全宗追杀
- 化神不出手干涉凡人事
- "魔功"修练会被正道围剿（但魔功见效快，所以总有人冒险）

## 主角处境
凝气 5 层的散修，在一线门派当外门记名弟子，无法升内门。
```

## 反面教材

❌ **创世神 + 八大种族 + 三千年史**（但故事只发生在一个村庄 1 个月）—— 设定过剩
❌ **规则前后矛盾**（前面说"杀同门必死"，后面主角杀了同门没事）—— 用 [asset-hygiene.md](./asset-hygiene.md) 改"已定"流程修
❌ **每条规则都解释清楚**（不留模糊地带）—— 留 20% 模糊给观众想象，故事更有空间
❌ **跟主线无关的国际局势**

## 用法

### 在 story-creator 工作流里
- **行为 ④ 升级项目化**触发时，按需创建 `世界观/<主题>.md`
- 单文件足够：`世界观/全部.md`（日常剧）
- 拆多文件：`世界观/规则.md` + `世界观/社会.md` + `世界观/地理.md`（玄幻 / 科幻）

### 跟 ai-video-prompt 衔接
- 世界观的"视觉签名"（如"灵气浓郁带有金色光雾"）会被 ai-video-prompt 抽到 prompt 的"风格"段
- 因此**视觉相关的设定要可拍**（"金色光雾"可拍，"灵气有 1.7 倍重力"不可拍）

### 跟 asset-index 联动
装了 CLI 后，世界观文件统一在 `世界观/` 目录，用 `asset-index search --type 世界观` 一次性看完。
