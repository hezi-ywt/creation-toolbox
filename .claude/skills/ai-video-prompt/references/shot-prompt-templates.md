# 分镜剧本模板（15s · 通用版）

> 状态：v0.1（10 个通用模板，题材中性，可套到不同题材）
> 默认时长：15 秒（Seedance 2.0 单次输出上限）
> 数据：基于 Seedance 2.0 调研 + 经典分镜原则整理
> 最后更新：2026-05-17

## 使用说明

**每个模板是 5 区块：**

1. **通用提示词** —— 这个模板的全局风格基线（一段中文）
2. **资产** —— 角色 + 场景（其他元素如道具 / 服装在剧本里描述）
3. **内容** —— 一段含画面感的剧情描述，读完心里有大致画面
4. **台词 / 音效** —— 对白 / 旁白 / 音效（带时间戳）
5. **分镜剧本** —— 喂 Seedance 的 Timeline 格式 prompt（唯一代码块）

**喂 Seedance 的方法：** 把「通用提示词」+「分镜剧本」拼起来喂进去。

**为什么不分中英两版：** 中文版 Seedance 直接能用，省一半篇幅。涉及运镜术语时分镜剧本里中英混着写（如"希区柯克变焦"/"dolly zoom"），关键术语英文括号保留也行。

**关于资产：** 模板默认只列**角色 + 场景**。道具 / 服装等细节直接在「内容」和「分镜剧本」里描述。项目实际启动时若涉及反复出现的关键道具，可在资产里追加。

**关于 beat 数：** 15s 默认 4 beat（[0s][4s][8s][12s]），但有些模板更适合 3 beat（如揭示式）或 5 beat（如复杂动作戏），不强求统一。

## 模板 01：三段式开场（15s · 3 beat）

### 通用提示词

电影感开场质感，大气广阔，慢节奏沉浸，自然光过渡，专业调色，构图严谨，画面稳定，2K 高清

### 资产

- **角色**：年轻女孩 · 18 岁 · 朴素长裙 · 黑色长发
- **场景**：山区小镇客栈窗前 · 黎明 · 远山云海

### 内容

黎明时分，山区小镇笼罩在淡淡晨雾中，远处群山被金色朝霞勾边。镜头先稳定地展示这片山水，然后缓慢推近到一座旧木质客栈的二楼窗前——一位身着素色长裙的年轻女孩正凭窗眺望远山，神情沉静中藏着复杂心事。最后切到她侧脸特写，捕捉她眼神里一闪而过的某个决心。

### 台词 / 音效

- **对白**：无
- **旁白 / 字幕**：无
- **音效**：环境（晨风 · 远处鸟鸣 · 木屋轻响）

### 分镜剧本

```
[0s] 极远景静止，俯瞰山区小镇全貌，淡晨雾笼罩，远山被金色朝霞勾边，
     地面有薄雾流动。摄影机完全不动，构图严谨，三层纵深感（前景屋顶 +
     中景街道 + 远景山脉）。

[5s] 缓慢推镜，从全景平滑推近至一座旧木质客栈的二楼窗户。窗后能依稀
     看到一位长发女孩的轮廓。节奏匀速，浅景深逐渐显现，暖金色晨光。

[10s] 特写女孩侧脸，正凭窗眺望远山。她的眼神从沉静转为一闪而过的决心。
      静止机位，柔和侧光勾勒五官，浅景深背景虚化成金色光斑。
```

## 模板 02：揭示式 / Reveal（15s · 2 beat）

### 通用提示词

戏剧化反差，悬念冲击，长镜头质感，视觉张力强，专业调色，画面稳定

### 资产

- **角色**：婴儿 · 6 个月大 · 干净的白色襁褓
- **场景**：末日废墟城市 · 黄昏 · 风沙弥漫

### 内容

镜头从一个无忧无虑、咯咯发笑的婴儿面部特写开始——温暖、纯净，让人以为是普通的家庭场景。然后镜头开始持续地拉远（dolly out），慢慢揭示婴儿其实躺在一栋崩塌建筑的瓦砾中央，周围是断裂的钢筋和燃烧后的废墟，远处城市天际线只剩残骸轮廓。婴儿是这片废墟中**唯一活着的存在**。

### 台词 / 音效

- **对白**：无
- **旁白 / 字幕**：无
- **音效**：婴儿咯咯笑声（持续）· 远处崩塌的金属呻吟 · 风沙呼啸

### 分镜剧本

```
[0s] 婴儿面部特写，纯净大眼睛、咯咯发笑，柔和暖光照亮粉嫩肌肤，
     浅景深，背景模糊。让观众以为是温馨家庭场景。摄影机静止 3 秒。

[3s] 开始缓慢拉远（slow dolly out），持续 12 秒。镜头从婴儿面部特写
     →中景显示白色襁褓和周围瓦砾→远景揭示婴儿躺在崩塌建筑中央
     →极远景显示整座废墟城市的末日景象。色调从温暖逐渐变冷峻，
     笑声仍在持续，反差极致。最终定格在婴儿独自在废墟中央的极远景。
```

## 模板 03：紧张铺垫（15s · 4 beat）

### 通用提示词

电影感，35毫米胶片颗粒，悬疑惊悚氛围，褪色黄昏色调，缓慢推进节奏，浅景深为主，高对比阴影，画面稳定，构图严谨

### 资产

- **角色**：男主 · 45 岁中年 · 深色西装外套 · 疲惫眼神
- **场景**：城市边缘空旷停车场 · 黄昏 · 寂静

### 内容

黄昏，城市边缘的空旷停车场，男主拖着疲惫的步伐走向远处停着的旧黑车，长投影斜跨水泥地。他注意到挡风玻璃下夹着一张折叠的白色纸条，脚步放慢，犹豫片刻后取下、缓慢展开。镜头从俯视全景一路推近，最终在希区柯克变焦的眩晕中定格在他瞳孔骤然收缩的瞬间——纸条颠覆了他对生活的某种认知。不揭示纸条内容，留给下一场。

### 台词 / 音效

- **对白**：无
- **旁白 / 字幕**：无
- **音效**：环境（远车流 · 风 · 皮鞋踩水泥地）· [8s] 纸条脆响 · [12s] 男主吸气

### 分镜剧本

```
[0s] 高角度远景，一位身穿深色西装的中年男子从画面左下走向停在中央的黑色轿车。
     黄昏金色光线，长投影斜跨水泥地面，远处传来车流环境音。摄影机静止，三层
     纵深构图。

[4s] 缓慢推镜，从背后跟随男子推近至中景。他注意到挡风玻璃下夹着一张折叠的纸条，
     脚步放慢。节奏减速，浅景深，暖金色侧光。

[8s] 特写男子粗糙的双手取下纸条、缓慢展开。清脆的纸张声。戏剧化黄昏轮廓光
     勾勒纸边，背景模糊，摄影机静止。

[12s] 极特写男子的眼睛，希区柯克变焦（眩晕效果）3 秒。瞳孔骤然收缩，背景轻微
      扭曲。虹膜上反射黄昏金光，伴随轻微吸气声。
```

## 模板 04：追逐戏（15s · 4 beat）

### 通用提示词

紧迫感强，纪实手持质感，高对比霓虹光，快节奏剪辑感，色彩饱和，动感模糊适度，画面稳定中带呼吸感

### 资产

- **角色**：男主 · 25 岁年轻人 · 黑色连帽卫衣 · 喘息中的紧绷脸 / 追逐者 · 西装暗色身影
- **场景**：拥挤的夜市街道 · 深夜 · 霓虹与雨水反光

### 内容

深夜，拥挤喧闹的亚洲风格夜市，霓虹招牌在湿润街道上投下斑斓倒影。男主穿着连帽卫衣狂奔，在小贩摊位和人群之间穿梭闪躲，呼吸急促。镜头紧贴他背后用手持跟拍，画面剧烈晃动，霓虹光柱飞驰而过。突然一个 whip pan 急摇向街道另一头，揭示追逐者正从人群中逼近。最后切回男主面部特写，他回头看到追逐者的瞬间，眼神瞬间放大。

### 台词 / 音效

- **对白**：男主急促喘息声
- **旁白 / 字幕**：无
- **音效**：环境（夜市喧嚣 · 小贩叫卖 · 雨声）· 男主奔跑脚步 · [9s] 急摇时声音被压低制造紧张 · [13s] 男主吸气

### 分镜剧本

```
[0s] 手持跟拍，从男主背后约 1 米跟随他在拥挤夜市中狂奔。画面自然抖动，
     霓虹光柱（粉色、青色、橙色）在湿润街道上拖出光斑，人群从画面两侧
     飞驰而过。男主胸部以上构图，紧绷的肩膀线条清晰。

[5s] 男主突然侧身闪过一个挑担小贩，镜头跟随急转。手持抖动加剧，
     运动模糊适度，氛围紧迫。

[9s] 急摇（whip pan）向街道另一头 1 秒，画面运动模糊。摇到另一头时
     清晰呈现一个身穿暗色西装的追逐者从人群中逼近，神情冷峻。

[12s] 切回男主面部特写，他正回头看到追逐者。瞳孔放大，呼吸停顿。
      手持轻微抖动，霓虹光在他脸上闪烁，戏剧化高对比阴影。
```

## 模板 05：抽离结尾（15s · 3 beat）

### 通用提示词

情感升华，温暖余韵，电影质感片尾感，慢节奏抽离，柔和光线，构图严谨，画面稳定

### 资产

- **角色**：老兵 · 70 岁 · 旧军绿色军装 · 胸前勋章
- **场景**：山间国家公墓 · 暮色 · 风吹动彩旗

### 内容

暮色中的山间国家公墓，整齐的白色墓碑在斜阳下投下长影。一位身穿旧军装的老兵站在一座特定的墓前，缓缓抬起颤抖的右手，向墓碑敬了一个庄严的军礼，眼眶湿润。镜头先紧贴他的面部特写捕捉这个瞬间，然后机位缓慢升起，拉远到他在整个公墓中孤独的小小身影，最后变成航拍俯瞰整片墓园在暮色中的全貌，金色彩旗在风中轻轻飘动。

### 台词 / 音效

- **对白**：无
- **旁白 / 字幕**：无
- **音效**：环境（晚风 · 远处国旗飘动声）· [0s] 老兵深呼吸 · [4s] 庄严的远方号角声渐起 · 整段配低沉弦乐

### 分镜剧本

```
[0s] 老兵面部特写，他正缓慢抬起颤抖的右手敬礼。皱纹深刻、眼眶湿润，
     胸前勋章在斜阳下闪光。柔和暖色侧光勾勒五官，浅景深背景墓碑模糊。
     静止机位 4 秒。

[4s] 缓慢升镜（slow crane up）+ 同时拉远，机位从老兵眼前升至他头顶
     上方约 5 米处。他在画面中逐渐变小，整齐排列的白色墓碑开始大面积
     进入画面。节奏匀速悠长。

[10s] 切换到航拍俯瞰，老兵成为整片山间公墓中央一个孤独的小小身影。
      暮色金光铺满墓园，金色彩旗在风中轻飘。镜头继续缓慢上升，画面
      最终定格在整个山谷的远景，远处青山如黛。
```

## 模板 06：关键道具引入（15s · 3 beat）

### 通用提示词

悬疑神秘氛围，戏剧化聚焦，浅景深为主，静谧紧张，暖色复古色调，画面稳定

### 资产

- **角色**：无（纯道具戏，或可暗示一个画外角色）
- **场景**：古宅书房 · 深夜 · 油灯昏黄

### 内容

深夜的古宅书房，月光从雕花木窗斜射进来，与桌上一盏老旧油灯的暖光交织。书桌上摆着一本封皮磨损的旧书，灰尘在光柱中飘浮。镜头先平静地呈现书房全貌——书架、椅子、烛台都清晰可见。然后焦点突然从背景的整体景观切到前景的那本旧书，仿佛某种意志在引导观众"注意这本书"。镜头随即缓慢推近，最终定格在书的封面，烫金书名在油灯光下若隐若现。

### 台词 / 音效

- **对白**：无
- **旁白 / 字幕**：无
- **音效**：环境（远处时钟滴答 · 风吹窗棂的轻响）· [5s] 焦点切换时音效有微妙下沉 · [10s] 推近时纸张轻微翻动声

### 分镜剧本

```
[0s] 静止远景，古宅书房全貌：左侧雕花木窗透月光，右侧高大书架，
     中央书桌上一盏老旧油灯散发暖光。所有元素均匀清晰，深景深，
     灰尘颗粒在光柱中漂浮可见。摄影机完全静止 5 秒。

[5s] 拉焦（rack focus），焦点从背景的书架和窗户切换到前景书桌上
     的一本旧书。背景突然虚化成柔和暖光，书的封面纹理变得锐利。
     摄影机仍静止，浅景深突出主体。

[10s] 缓慢推镜（slow dolly in），从书桌前的中距离平稳推近至旧书的
      封面极特写。烫金书名在油灯光下若隐若现，封皮磨损的纹理纤毫
      毕现。结束时定格 1 秒，给观众"注视"时间。
```

## 模板 07：戏剧化亮相（15s · 4 beat）

### 通用提示词

英雄式登场质感，戏剧化氛围，史诗光影，环绕动感，浅景深主体清晰，专业商业大片调性

### 资产

- **角色**：武士 · 30 岁 · 黑色和服 · 长发束起 · 腰挎武士刀
- **场景**：晨雾竹林深处 · 黎明 · 薄雾翻涌

### 内容

黎明时分，深山竹林笼罩在浓厚晨雾中，斜射的金色阳光穿透竹叶形成神秘光柱。一位身穿黑色和服的武士背对镜头静静站立在竹林中央，长发束起，腰挎武士刀，氛围庄严肃穆。镜头先静止 4 秒建立这个画面，然后缓慢环绕（orbit）武士移动，从背影逐渐转到侧脸、再到正脸，期间镜头同时缓慢升起，最终定格在武士冷峻眼神的面部特写——那是已经做好决断的眼神。

### 台词 / 音效

- **对白**：无
- **旁白 / 字幕**：无
- **音效**：环境（晨风 · 竹叶轻响 · 远处水滴）· [8s] 远处低沉鼓声渐起 · [12s] 武士刀微微出鞘的金属声

### 分镜剧本

```
[0s] 静止中景，武士背影立于竹林中央，黑色和服与浅色竹林形成对比。
     金色阳光穿透竹叶形成神秘光柱，浓雾在他脚边翻涌。摄影机完全
     静止 4 秒，三层纵深构图（前景竹叶 + 中景武士 + 远景雾林）。

[4s] 开始环绕镜头（orbit），围绕武士做半圆形运动，从背影转到侧面。
     节奏缓慢匀速，背景竹林持续旋转，武士保持画面中心。逆光勾边
     效果显著。

[8s] 环绕继续，同时缓慢升起（rising orbit），从平视转为俯视。武士
     正脸逐渐显现，冷峻沉稳的表情清晰可见。光线从侧逆变为顶逆，
     更增戏剧感。

[12s] 切到武士面部极特写，眼神冷峻锐利，已做好决断。静止机位，
      浅景深背景完全虚化，金色顶光在他眼睛里反射成两点星光。
```

## 模板 08：对话双人（15s · 4 beat）

### 通用提示词
